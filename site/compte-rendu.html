Présentation du Projet

VELORESTAU est une application web permettant de visualiser les stations de vélos et les restaurants à Nancy. Cette application a été développée dans le cadre du projet SAE.

Architecture du Projet

Technologies Employées

  • Frontend : HTML, CSS, JavaScript
  • Librairies : Lucide Icons, Leaflet.js
  • Backend : Java
  • Services : RMI, HttpServer (Proxy) et serviceDb (RMI)
  • Base de données : Oracle (Charlemagne)
  • Hébergement : Webetu

Schéma de la base de données

Réalisations

Services Implémentés

  • Service RMI pour la gestion des restaurants
    • Inscription au service central
    • Récupération des coordonnées des restaurants
    • Système de réservation de tables
  • Service de données ouvertes
    • Récupération des données des stations vélibs
    • Affichage des disponibilités en temps réel
  • Proxy HttpServer
    • Gestion des données bloquées
    • Accès aux incidents de circulation
    • Intégration avec le service RMI

Aspects Techniques

  • Gestion des dépendances avec Maven
  • Compilation automatique
  • Documentation Javadoc de l'API publique
  • Gestion des erreurs réseau et des timeouts
  • Configuration du proxy IUT

Guide d'Utilisation

Prérequis

  • Java 8 ou supérieur
  • Maven
  • Oracle Database (sur Charlemagne)
  • Un navigateur web moderne
